# npm

npm 최신 버전으로 업데이트 방법

- latest 버전 인스톨

```
npm install -g npm@latest
```

- next 버전 인스톨

```
npm install -g npm@next
```



# docsify

개발 환경 설정 (node.js 설치가 되어 있어야 함)

```
npm i docsify-cli -g
```

cd 명령어로 프로젝트 폴더로 이동

```
docsify init ./<프로젝트명>
```

생성 테스트

```
docsify serve ./<프로젝트명>
```

localhost:3000/ 에 접속해서 확인.
