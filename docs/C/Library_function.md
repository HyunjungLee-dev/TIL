#### isdigit 숫자를 판단하는 함수

isdigit 함수는 문자가 숫자 0~9 사이에 속하는지 검사하는 함수이다.

**헤더 파일**

#include <ctype.h> // c

#include <cctype> // c++

함수 원형은 int isdigit(int c); 

문자가 0~9 사이에 속하면 true 아니면 false
