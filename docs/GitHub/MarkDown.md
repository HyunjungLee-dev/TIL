## 마크다운 텍스트/백그라운드 색상 넣기

기본적으로 마크다운 문법에서는 색상 넣는 것을 지원하지 않기 때문에 HTML 태그를 이용해야한다.

```html
<span style="color:red">Span Text</span> // 글자색 지정
<span style="background-color:red">Span Text</span> // 배경색 지정
```
